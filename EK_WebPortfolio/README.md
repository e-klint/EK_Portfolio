# Webbportfolio för webbutvecklare

## Om Projektet
En personlig webbportfolio som är byggd med HTML, CSS och JavaScript från grunden utan ramverk. 
Det är en sida (SPA) som följer en klassisk layout med flera sektioner som innehåller 
bl.a. projekt, referenser och kontaktformulär. Syftet är att presentera vem jag är, 
mina projekt och mina kunskaper. 

Webbsidan är användarvänlig och responsiv, dvs. anpassad till desktop, surfplatta och mobil.

## Features
- Responsiv layout med Flexbox
- **Navigationsmeny**
- **Om mig-sektion:**
- Porträttbild
- Knapp för att ladda ned CV
- **Kunskaper/Referenser-sektion:**
- **Portfölj-sektion med projektlänkar**
- **Kontakt-sektion:**
- Kontaktformulär
- Formulärvalidering med felmeddelanden
- **Footer**
- Länkar till sociala medier och GitHub
- Väderwidget som använder ett externt API

## Tekniker
- HTML
- CSS
- Flexbox
- JavaScript
- Back-end tjänst: FormSubmit [https://formsubmit.co/]
- Väder API (OpenWeather): [https://openweathermap.org/api]
- Git och github

## Installation
Kör projektet lokalt i VS Code.

1. Klona repot
2. Öppna index.html-filen i webbläsaren.

## Best Practices
- Semantisk HTML-struktur
- Responsive Design
- Separation of concerns (HTML, CSS och javascript är separerade.
- CSS är uppdelad i flera filer: base, layout, components, utilities och responsive. Återanvändbara klasser.
- Input validation
- Git commits med feature branches och tydliga meddelanden.

## Responsiv design
- Media Queries
- Flexibel layout (Flexbox)
- Mobilvänlig navigation
